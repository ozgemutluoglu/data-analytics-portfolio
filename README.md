## Projects

### 🛒 [FoodYum Grocery Store — SQL Data Cleaning & Analysis](./foodyum-grocery-sales)
A PostgreSQL project focused on cleaning a messy grocery product dataset (missing values, inconsistent text formatting, mixed units) and analyzing pricing patterns across product categories.
**Skills:** `COALESCE`, `PERCENTILE_CONT` (median imputation), string cleaning, CTEs, aggregate functions.

### 🎮 [SaaS Game Metrics Dashboard — MRR, Churn & LTV Analysis](./saas-game-metrics-dashboard)
An interactive Tableau Public dashboard analyzing monthly cash flow, revenue dynamics, and user retention across 3 mobile games, powered by a chained-CTE SQL pipeline.
**Skills:** Window functions (`LAG`), cohort-based churn calculation, MRR/ARPPU/LTV modeling, Tableau Public.
