## Projects

### 🛒 [FoodYum Grocery Store — SQL Data Cleaning & Analysis](https://github.com/ozgemutluoglu/data-analytics-portfolio/blob/main/foodyum-grocery-sales)

A PostgreSQL project focused on cleaning a messy grocery product dataset (missing values, inconsistent text formatting, mixed units) and analyzing pricing patterns across product categories. **Skills:** `COALESCE`, `PERCENTILE_CONT` (median imputation), string cleaning, CTEs, aggregate functions.

### 🎮 [SaaS Game Metrics Dashboard — MRR, Churn & LTV Analysis](https://github.com/ozgemutluoglu/data-analytics-portfolio/blob/main/saas-game-metrics-dashboard)

An interactive Tableau Public dashboard analyzing monthly cash flow, revenue dynamics, and user retention across 3 mobile games, powered by a chained-CTE SQL pipeline. **Skills:** Window functions (`LAG`), cohort-based churn calculation, MRR/ARPPU/LTV modeling, Tableau Public.
🔗 [Live dashboard](https://public.tableau.com/views/NakitAkDashboardOyunAnalizi2022/NakitAkDashboardOyunAnalizi2022?:language=en-US&:display_count=n&:origin=viz_share_link)

### 🌍 [World Happiness Report 2015–2019 — pandas Merge, EDA & Tableau Dashboard](https://github.com/ozgemutluoglu/data-analytics-portfolio/blob/main/world-happiness-report-2015-2019)

A 5-year, 782-row dataset built by merging five yearly World Happiness Report CSVs with inconsistent schemas in pandas, followed by correlation analysis and an interactive, year-filtered Tableau dashboard. Team project (4 collaborators). **Skills:** `pandas` (schema alignment, `concat`, country-based backfill), correlation analysis (pooled vs. per-year pitfalls), `matplotlib`/`seaborn`, Tableau Public.
🔗 [Live dashboard](https://public.tableau.com/views/WORLDHAPPINESSREPORT2015-2019_17821369701280/Dashboard1?:language=en-US&:display_count=n&:origin=viz_share_link)
